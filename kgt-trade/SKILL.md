# 輸出入・貿易専門エージェント（KGT グループ共通）

タイ日間を中心とした精密部品・産業材料の輸出入実務知識を提供します。
通関・関税・書類作成の実務は通関士・フォワーダーと連携して進めます。

---

## 1. インコタームズ 2020（Incoterms®2020）

```
リスクと費用の移転ポイントを定めた国際規則（ICC が制定）

よく使われる条件（製造業向け）:

EXW（Ex Works：工場渡し）:
  売主: 梱包のみ。工場/倉庫で引渡し
  買主: 積込みから全て負担
  注意: 輸出通関も買主責任 → 実務上は不便

FOB（Free On Board：本船渡し）:
  売主: 指定積出港で本船船上に積み込むまで
  買主: 本船積込み後から全て負担
  使用場面: 海上輸送（航空貨物には CPT/CIP を推奨）

CIF（Cost, Insurance and Freight）:
  売主: 仕向港までの運賃・保険料を負担
  買主: 仕向港での輸入通関・国内輸送
  注意: 保険受益者は買主なのに売主が手配 → CIP の方が望ましい

DAP（Delivered at Place）:
  売主: 指定仕向地（買主倉庫等）への搬入まで負担
  買主: 輸入通関・関税・VAT
  使用場面: ドアデリバリーが必要な場合

DDP（Delivered Duty Paid）:
  売主: 輸入通関・関税まで全て負担
  買主: 荷下ろし後のみ
  注意: 売主のリスク・コスト最大 → 慎重に使う

推奨の使い方（KGT）:
  タイ→日本輸出: FOB Bangkok または CIF 日本港
  日本→タイ輸入: FOB 日本港 または CIF Bangkok
```

## 2. HS コード（関税分類）

```
Harmonized System（HS）: 国際的な商品の分類コード体系

コードの構造（例: 8483.40.10）:
  84    → 類（Chapter）: 原子炉・ボイラー・機械
  8483  → 項（Heading）: トランスミッション・シャフト
  8483.40 → 号（Subheading）: 歯車・ギヤ・ボール
  8483.40.10 → タイ/日本固有の細分（国内細目）

精密プラスチック部品の主な HS コード（参考）:
  3926.90 → プラスチック製品（その他）
  8477.xx → プラスチック加工機械
  8484.xx → ガスケット・シール
  9031.xx → 測定・検査機器

注意:
  HS コードは輸出国と輸入国で一致しない場合がある
  → フォワーダーまたは通関士に事前確認必須
  → 誤った HS コードは過徴収/脱税リスク
```

## 3. タイ→日本 輸出の実務

### タイ側の輸出手続き

```
必要書類（一般的なケース）:
  □ Commercial Invoice（商業インボイス）
  □ Packing List（梱包明細）
  □ Bill of Lading（船荷証券）or Air Waybill（航空運送状）
  □ Certificate of Origin（原産地証明書）← 関税優遇のため
  □ Export Declaration（輸出申告書）

原産地証明書（タイ→日本）:
  JTEPA（日タイ EPA）優遇関税の適用に必要
  → タイ商務省 Department of Foreign Trade へ申請
  → Form JT（JTEPA 用フォーム）を使用
  → 関税が大幅削減される品目あり（要確認）

輸出禁止・規制品目（タイ）:
  文化財・抗生物質・特定化学品等
  精密部品は原則規制なし（軍事転用可能品は除く）
```

### 日本側の輸入手続き

```
輸入通関（日本税関）:
  輸入申告 → 審査・検査（必要に応じ）→ 関税・VAT 納付 → 輸入許可

輸入関税率（参考、2026年時点）:
  プラスチック製品: 0〜6.6%（品目により異なる）
  JTEPA 適用: 多くの品目で 0%
  → 原産地証明書（Form JT）の添付で JTEPA 税率適用

消費税（輸入時）:
  課税価格（CIF価格）× 10%
  輸入者が立替払い → 仕入れ税額控除（後で還付）

輸入規制:
  食品衛生法（樹脂製品が食品に触れる場合）
  化審法（新規化学物質）
  → 該当する場合は事前確認必須
```

## 4. 日本→タイ 輸入の実務

### 日本側の輸出手続き

```
経済産業省への輸出許可（該当品目）:
  リスト規制: 規制品目は許可申請必須
  キャッチオール規制: 大量破壊兵器等への転用懸念時

輸出通関書類:
  □ Commercial Invoice
  □ Packing List
  □ Bill of Lading / Air Waybill
  □ 原産地証明書（必要な場合）
  □ 輸出申告書（輸出許可書）
```

### タイ側の輸入手続き

```
タイ関税局（Customs Department）への申告

輸入関税（タイ）:
  機械・設備: 0〜10%（品目による）
  プラスチック原料: 1〜5%
  完成品: 10〜30%（品目による）
  JTEPA 適用: 多くの品目で削減または免除

VAT（輸入時）:
  課税価格（CIF + 関税）× 7%

BOI 優遇企業の場合:
  BOI 対象機械・設備は輸入関税免除
  原材料は BOI 認定を受けた場合に免除

通関業者（Customs Broker）の利用:
  タイでは通関業者（Shipping Agent）を通すのが一般的
  費用: 1件あたり 500〜3,000 バーツ程度
```

## 5. 貿易書類の書き方

### Commercial Invoice（商業インボイス）

```
必須記載事項:
  1. 発行日・インボイス番号
  2. 売主情報（会社名・住所・TAX ID）
  3. 買主情報（会社名・住所）
  4. 品名（英語で詳細に）
  5. HS コード（推奨）
  6. 数量・単位
  7. 単価・金額（通貨明示）
  8. インコタームズ条件
  9. 支払条件（T/T 30日・L/C等）
  10. 原産国（Country of Origin）

注意:
  □ 過少申告（Undervaluation）は厳禁（脱税・罰則）
  □ ギフト扱いで低価格記載も不可
  □ サンプル（無償）の場合は "No Commercial Value" を明記
     ただし税関が認めない場合もある → フォワーダーに確認
```

### Packing List（梱包明細）

```
記載事項:
  1. 品名・型番
  2. 個数・数量
  3. 梱包単位（外箱数・内容量）
  4. 寸法（L×W×H cm）
  5. 重量（Gross Weight / Net Weight）
  6. マーキング（荷印）

精密部品の特記事項:
  □ 梱包材（防湿・防振）の記載
  □ 向き指示（This Side Up）
  □ 積み重ね禁止（Do Not Stack）等
```

## 6. 支払方法

```
T/T（電信送金）:
  最も一般的。T/T30日（出荷後30日以内支払い）等
  リスク: 代金未払いリスク（信用取引）

L/C（信用状）:
  銀行が支払いを保証
  新規顧客・大口取引・政情不安な国向けに有効
  コスト: L/C 開設手数料（0.1〜0.5%程度）

D/P（支払渡し）/ D/A（引受渡し）:
  銀行経由で書類と引換に代金回収
  D/P: 代金支払い後に船荷証券を交付
  D/A: 支払約束（手形引受）後に書類交付

為替リスク管理:
  THB/JPY レートの変動が利益を左右
  → 長期契約時はドル建て（USD）を検討
  → ヘッジ（先物・オプション）は銀行と相談
```

## 7. フォワーダーとの協業

```
フォワーダー（貨物利用運送事業者）の役割:
  - 輸出入通関手続きの代行
  - 海上/航空スペースの手配
  - 梱包・ラベリングのアドバイス
  - 書類作成支援

選定基準:
  □ タイ↔日本ルートの実績
  □ 通関士資格の保有
  □ 緊急対応（航空への切替等）の可否
  □ 価格（海上 vs 航空の比較見積）

リードタイム目安（タイ→日本）:
  海上: 10〜15日（Bangkok → 横浜/東京/名古屋）
  航空: 2〜3日（Suvarnabhumi → 成田/関空）
  → 航空は海上の5〜10倍のコスト
```

## 8. 輸出入 KPI

```
貿易コスト管理:
  □ 関税コスト / 売上高（目標: JTEPA 活用で最小化）
  □ 輸送費 / 売上高（FOB 基準で管理）
  □ 通関遅延日数（目標: ゼロ）

書類管理:
  □ インボイス・パッキングリストの電子保存
  □ 原産地証明書の有効期限管理（通常12ヶ月）
  □ L/C の有効期限・船積期限の管理

コンプライアンス:
  □ 禁輸国・制裁リストとの照合（毎取引）
  □ HS コードの定期レビュー（品目変更時）
  □ BOI 輸入原料の数量管理（年次報告義務）
```
