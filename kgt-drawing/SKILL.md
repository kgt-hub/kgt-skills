# 図面解釈専門エージェント（KGT グループ共通）

機械製図の国際規格（JIS/ISO）に基づく図面読み取りと品質管理への応用知識を提供します。
プロジェクト固有のコードやテーブルへの依存なく適用できます。

## 1. 製図規格

```
JIS B 0001: 機械製図（一般規則）
JIS B 0405: 普通公差（個々に公差が指示されていない長さ寸法・角度）
JIS B 0419: 幾何公差（独立原則・エンベロープ要求事項）
JIS B 0601: 製品の幾何特性仕様（GPS）— 表面性状
ISO 1101:   幾何公差表記法（国際標準）
ISO 2768:   普通公差（寸法・形状）
ISO 1302:   表面性状の表示方法
```

### 個別公差（直接指示）の読み方

```
φ25.000 +0.010/-0.000 の意味:
  名目値: φ25.000mm
  上限偏差: +0.010mm（最大 25.010mm）
  下限偏差: -0.000mm（最小 25.000mm）

表記の種類:
  両側公差: φ25 ±0.05   → 24.95 〜 25.05
  片側公差: φ25 +0.02/0  → 25.00 〜 25.02
  限界寸法: φ25.02/24.98 → 24.98 〜 25.02

記号の意味:
  φ（Φ）: 円形断面の直径
  □:      正方形断面の一辺
  R:      半径（= 直径 / 2）
  SR:     球の半径
```


### 公差域記号フレームの読み方

```
|◎|φ0.05|A|B| の意味:
  ◎: 位置度
  φ0.05: 公差値 0.05mm の円柱公差域
  A, B: データム（基準面）

|⊥|0.02|A| の意味:
  ⊥: 直角度
  0.02: 公差値 0.02mm の平行平面公差域
  A: データム A 面に対して
```

## 4. 表面粗さ

### 表面粗さパラメータ

```
Ra（算術平均粗さ）: 最も一般的に使われる指標
  → 測定長さにわたる凹凸の算術平均値

Rz（最大高さ粗さ）: 最大の山谷間距離
  → 旧 JIS の Rmax に相当

一般的な目安:
  Ra 0.1以下  → 超精密仕上げ（ラップ・研磨）
  Ra 0.4     → 鏡面仕上げ（研削）
  Ra 0.8     → 精密仕上げ（精密研削）
  Ra 1.6     → 仕上げ（精削り）
  Ra 3.2     → 並仕上げ（通常の旋削・フライス仕上げ）
  Ra 6.3     → 荒仕上げ（荒削り）
  Ra 12.5    → 粗仕上げ（粗削り）
  Ra 25以上  → 加工面そのまま

表面粗さ記号の読み方（ISO 1302）:
  √ Ra1.6: Ra ≦ 1.6μm で仕上げる
  = （表面テクスチャ除去加工不可）
  ○ （加工方法を問わない）
```

### プラスチック部品の表面粗さ注意点

```
PA（ナイロン）・PC（ポリカーボネート）:
  Ra 0.8以下 → 研削加工が必要（外注を検討）
  Ra 1.6     → 精密仕上げ切削で達成可能
  Ra 3.2以上 → 通常切削で容易に達成

注意: プラスチックの Ra は吸湿・温度変化で変化する
     精密な表面粗さ管理が必要な場合は温度・湿度管理下で測定
```

## 7. よくある図面読み取りミスと対策

| ミス | 原因 | 対策 |
|---|---|---|
| 普通公差と個別公差の混同 | 両方が同じ図面に存在する | 公差枠（タイトルブロック）を先に確認 |
| Ra と Rz の混同 | 旧 JIS との移行期の混在 | 表面粗さ記号の直後の文字を確認 |
| 幾何公差のデータム見落とし | 複雑な図面で後回し | データム文字（A・B・C）を先にマーク |
| 公差の符号逆（+と-を間違い） | 締まり嵌め/すき間嵌めの勘違い | 嵌め合いの種類（H7/g6 等）を確認 |
| プラスチックへの金属公差適用 | 慣習的な値の流用 | PA/PC 等の吸湿変形量を加算して設定 |
| 膜厚処理後の寸法変化見落とし | 設計段階で処理代未考慮 | メッキ・アルマイト膜厚を加工前に引く |
| 材料グレード不足認識 | PA6 と PA66 の区別等 | 材料欄の完全な記号を確認・確認書取得 |
| 角度公差の見落とし | 長さ寸法に集中しすぎ | 角度・テーパー部分を独立してレビュー |

## 8. 嵌め合い（Fits）

```
ISO/JIS の嵌め合い記号:

穴基準（穴は H、軸が変わる）:
  H7/g6: すき間嵌め（軸受・摺動部）
  H7/k6: 中間嵌め（軽圧入・位置決め）
  H7/p6: しまり嵌め（圧入固定）

軸基準（軸は h、穴が変わる）:
  D8/h7: すき間嵌め
  K7/h6: 中間嵌め
  P7/h6: しまり嵌め

プラスチック部品での注意:
  吸湿による膨張でしまり嵌めが過大になる可能性
  → 嵌め合い設計時は吸湿膨張量を考慮した公差設定
```
