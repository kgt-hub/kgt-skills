## 1. プラスチック切削加工の基本原則

### 金属切削との決定的な違い

```
熱変形:
  プラスチックは熱伝導率が低く切削熱が逃げにくい
  → 切削熱が蓄積して変形・寸法精度悪化
  → 低速または少量切込みで熱を抑制

弾性変形（スプリングバック）:
  切削中に材料がたわみ、工具通過後に戻る
  → 薄肉部品・長物の最終仕上げで要注意
  → ワーク支持・バックアップが重要

吸湿変形（PA・PC等）:
  加工後に吸湿して寸法が変化する
  → 加工→安定→測定の時間管理が必要

クランプ変形:
  金属と同じ力でクランプすると変形する
  → 軟質材（PTFE・PE）は特に注意
  → 補助治具・分散支持で対応
```

## 3. 工具選定

```
すくい角（Rake Angle）:
  プラスチック: 正のすくい角（10〜20°）推奨
  → 切削抵抗を下げ、切削熱を軽減
  → 切れ刃がシャープであることが重要

逃げ角（Clearance Angle）:
  プラスチック: 5〜15°（金属より大きめ）
  → 摩擦による切削熱を軽減

工具材質の推奨:
  汎用プラスチック（POM・PP・ABS）: HSS も使用可
  エンプラ（PA・PC・POM）: 超硬推奨
  スーパーエンプラ（PEEK）: 超硬必須
  繊維強化（GFRP・CFRP）: ダイヤコーティング必須
  PTFE: 高速度鋼（鋭利仕上げ）または超硬

切削油・冷却材:
  多くのプラスチック: 乾式またはエアブロー推奨
  PA・PC: 水系切削液は吸湿の原因 → 使用禁止
  PEEK: 乾式推奨
  POM・ABS: 水溶性切削液も使用可（ただし乾式が基本）
```

## 4. 熱リスク判定

```python
def check_heat_risk(material_code: str, cutting_speed_m_min: float) -> str:
    HEAT_SENSITIVE = {
        "PP":  130,  # 変形開始温度（°C）
        "PE":  120,
        "ABS": 100,
        "PC":  145,
        "PA6": 180,
        "PA66": 200,
        "POM": 165,
        "PTFE": 180,
        "PEEK": 300,
    }
    threshold = HEAT_SENSITIVE.get(material_code, 150)
    # 切削速度が高いほど切削熱が増加（簡易判定）
    if material_code in ["PP", "PE", "ABS"] and cutting_speed_m_min > 200:
        return "🔴 切削熱変形リスク高 → 速度を下げてエアブローを追加"
    elif material_code == "PC" and cutting_speed_m_min > 150:
        return "🟡 PC の応力亀裂リスク → 切削速度 150m/min 以下に"
    elif material_code == "PTFE" and cutting_speed_m_min > 150:
        return "🟡 PTFE の変形リスク → 150m/min 以下に抑える"
    return "🔵 問題なし"
```

## 5. 加工工程設計

```
一般的な精密プラスチック部品の工程:

1. 素材確認（材料証明書・ロット番号）
2. 乾燥処理（PA・PC は必須）
   → PA6: 80°C × 4h / PC: 120°C × 4h
3. 粗加工（加工代 0.5〜1.0mm 残し）
4. 応力緩和待ち（必要な場合: 数時間〜24h）
5. 仕上げ加工
6. 寸法検査（加工後 24h 以上経過後が望ましい）
7. 外観検査
8. 梱包（PA・PC は防湿梱包推奨）

NC 加工のポイント:
  □ G00 早送り時のワーク衝突注意（材料が動きやすい）
  □ 仕上げ切込みは小さく（0.1〜0.3mm）
  □ 工具パスは一方向仕上げを検討（面粗さ向上）
  □ コーナーR の指定（応力集中を避ける）
```

## 6. 寸法安定性管理

```
問題: 加工直後の測定値が翌日変わっている
主な原因:
  1. 吸湿（PA・PC）→ 寸法が増加
  2. 切削応力の緩和 → わずかに変形
  3. 温度変化による熱膨張

対策:
  □ 精度が必要な寸法は加工後 24h 以降に測定
  □ 測定は温度管理された環境（23°C ±2°C）で
  □ PA 系は完成品を密封袋に入れて管理
  □ 加工環境の温湿度を記録（品質トレーサビリティ）

プラスチックの線膨張係数（参考）:
  POM:   10〜12 × 10⁻⁵ /°C
  PA6:   10〜15 × 10⁻⁵ /°C（吸湿でさらに増加）
  PC:    6〜7  × 10⁻⁵ /°C
  PEEK:  4〜5  × 10⁻⁵ /°C
  → 金属（SUS304: 1.7 × 10⁻⁵）の5〜10倍
  → ±0.05mm の公差は温度管理なしには難しい場合あり
```

## 7. 不良モードと対策

| 不良 | 原因 | 対策 |
|---|---|---|
| 寸法オーバー（吸湿） | PA/PC の吸湿膨張 | 防湿保管・加工後安定待ち |
| バリ | 切れ味不足・送り過多 | 刃先研磨・送り減少・最終パスを軽切込みに |
| 表面ムラ・鱗状 | 切削熱で表面溶融 | 速度下げ・エアブロー強化 |
| 変形（たわみ） | クランプ過多・薄肉 | クランプ力軽減・補助支持治具 |
| 内部クラック（PC）| 切削応力 + 溶剤接触 | クランプ力管理・溶剤接触禁止 |
| 工具寿命が短い（PEEK） | 硬度・繊維材 | 超硬/CBN 工具・早めの交換 |
| 粉っぽい切り屑 | 脆い材料または高速すぎ | 速度調整・切込み変更 |
| 長い連続切り屑 | 延性材料・送り過少 | 送り増加・チップブレーカ形状変更 |
