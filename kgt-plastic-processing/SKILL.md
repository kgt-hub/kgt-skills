## 1. 射出成形（Injection Molding）

### 基本プロセス

```
プロセス:
  1. 原料ペレットを乾燥（吸湿材料は必須）
  2. 加熱シリンダーで溶融
  3. スクリューで金型キャビティに射出
  4. 冷却・固化
  5. 型開き・成形品取り出し
  6. 後処理（バリ取り・外観検査）

サイクルタイムの内訳:
  射出時間: 1〜5秒
  冷却時間: 5〜60秒（材料・肉厚による）
  取り出し: 2〜5秒
  合計: 10〜70秒/ショット
```

### 射出成形の不良モード

```
短尺（ショートショット）:
  原因: 射出圧力不足・ゲート詰まり・材料不足
  対策: 射出圧力UP・温度UP・ゲート拡大

ヒケ（Sink Mark）:
  原因: 冷却速度の不均一・肉厚変化が急激
  対策: 冷却時間延長・保圧UP・リブ設計見直し

ウェルドライン（Weld Line）:
  原因: 複数ゲートからの溶融樹脂の合流
  対策: ゲート位置変更・温度UP・デザイン見直し

バリ（Flash）:
  原因: 過充填・クランプ力不足・金型摩耗
  対策: 射出圧力DOWN・型締め力UP・金型修理

反り（Warpage）:
  原因: 冷却不均一・残留応力
  対策: 均一冷却・取り出し後の矯正・設計変更

焼け（Burn Mark）:
  原因: 過熱・エアトラップ
  対策: 温度DOWN・エアベント設置・射出速度DOWN

気泡（Void / Bubble）:
  原因: 乾燥不足・揮発成分
  対策: 十分な乾燥・背圧調整
```

## 2. 押出成形（Extrusion）

```
用途: 棒材・チューブ・シート・フィルムの連続成形

プロセス:
  ペレット投入 → 溶融 → ダイス通過 → 冷却水槽 → 引取機 → 切断

断面形状:
  丸棒（Rod）: 旋削素材として多用
  パイプ（Tube）: 中空部品
  角棒（Profile）: フライス素材

押出材の特性:
  □ 射出成形品より均一な内部構造
  □ 押出方向に配向が生じる（異方性に注意）
  □ 外径は公差がゆるめ → 切削加工で仕上げる
```

## 3. ゴム加工プロセス

### ゴム成形の主要工法

```
圧縮成形（Compression Molding）:
  未加硫ゴムを金型にセット → 加熱加圧 → 加硫
  メリット: 金型が安価・大型成形可
  デメリット: バリが多い・成形サイクル長い
  用途: Oリング・パッキン・防振ゴム

射出成形（Transfer / Injection Molding）:
  溶融ゴムを金型に射出 → 加硫
  メリット: 精度高い・バリ少ない
  デメリット: 金型が高価
  用途: 精密シール・自動車ゴム部品

押出成形（Extrusion）:
  ゴムコンパウンドをダイスから押出 → 加硫
  用途: ホース・チューブ・ウェザーストリップ
```

### 加硫（Vulcanization）

```
加硫: 硫黄や過酸化物でゴム分子を架橋結合させる工程
  → 弾性・耐熱性・耐久性が大幅向上

加硫条件（例: NBR）:
  温度: 160〜180°C
  時間: 5〜30分（肉厚による）
  圧力: 5〜15 MPa

未加硫・過加硫の問題:
  未加硫: 軟らかすぎ・永久変形・耐久性不足
  過加硫: 硬くなりすぎ・亀裂発生
```

## 4. スポンジ・発泡材の加工

```
発泡プロセス（化学発泡）:
  発泡剤を混合 → 加熱で発泡剤分解・ガス発生 → 気泡形成

打ち抜き加工（Die Cutting）:
  シート状スポンジから必要形状を打ち抜き
  → ガスケット・パッキン・衝撃吸収材の製造

水切り加工（Water Jet Cutting）:
  複雑形状・精密カット
  → 発泡材を変形させずに切断可能

接着・貼り合わせ:
  □ 接着剤選定は材料との相性を確認
  □ PE・PP スポンジは接着しにくい → プライマー処理
  □ シリコーンスポンジは特殊接着剤が必要
```

## 6. 後処理工程

```
バリ取り:
  □ 手作業（ヤスリ・カッター）: 少量・複雑形状
  □ バレル研磨: 量産・均一処理
  □ 超音波洗浄後バリ取り: 精密部品
  □ 注意: プラスチックは柔らかいため傷つきやすい

表面処理:
  □ プラズマ処理: 接着性向上（PE・PPの接着前処理）
  □ コーティング: 耐摩耗・耐薬品（PTFE コーティング等）
  □ 染色: PA・PC は染色可能

熱処理（アニーリング）:
  残留応力の除去・寸法安定化
  PC: 120°C × 2〜4h（加工後）
  PA: 100°C × 2h in oil（エンジニアリング用途）
  PEEK: 200°C × 2h
```

## 7. 品質管理・検査

```
成形品の検査項目:

外観検査（全数）:
  □ ヒケ・バリ・ウェルドライン・焼け
  □ 表面キズ・異物混入
  □ 色むら・変色

寸法検査（抜取 or 全数）:
  □ 重要寸法の測定（ノギス・マイクロメータ・CMM）
  □ 機能確認（嵌め合い・組付け）

機械的特性（定期確認）:
  □ 引張強度・曲げ強度
  □ 硬度
  □ 衝撃強度

サンプル保管:
  □ ロット別に代表サンプルを保管
  □ 不良発生時のトレーサビリティ確保
```

## 8. ロット管理

```
重要性:
  品質問題発生時に原材料・成形条件・外注先を特定できる

管理項目:
  □ 原料ロット番号（材料メーカー証明書）
  □ 乾燥条件（温度・時間・担当者）
  □ 成形機番号・金型番号
  □ 成形日時・オペレーター
  □ 検査結果・判定

ロットトレーサビリティフロー:
  原料ロット → 乾燥処理記録 → 成形記録 → 検査記録 → 出荷記録
  → 不良発生時: どの原料・どの成形ロットかを追跡
```
