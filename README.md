# KGT グループ共通スキル（Claude Code 用）

KGT グループ全プロジェクトで共有する、製造業ドメイン知識の専門スキル集です。

## 使い方

各スキルディレクトリを `~/.claude/skills/` にコピーして使用します。

```bash
cp -r kgt-* ~/.claude/skills/
```

## スキル一覧

| スキル名 | 専門領域 |
|---|---|
| `kgt-legal` | 法務（タイ・日本の契約法・労働法・会社法・輸出規制） |
| `kgt-tax` | 税務・会計（タイ法人税・VAT・源泉税・移転価格・日本税務） |
| `kgt-trade` | 貿易・輸出入（インコタームズ・HS コード・通関・貿易書類） |
| `kgt-kaizen` | 改善活動（TPS・7つのムダ・5S・PDCA・ポカヨケ・OEE） |
| `kgt-drawing` | 図面解釈（JIS/ISO 製図・寸法公差・幾何公差・表面粗さ） |
| `kgt-iso` | ISO 品質マネジメント（ISO 9001・IATF 16949・FMEA・SPC・MSA） |
| `kgt-machining` | プラスチック切削加工（切削条件・工具選定・寸法安定性） |
| `kgt-plastic-materials` | 工業材料（エンプラ・汎用樹脂・ゴム・スポンジ・コンベアベルト） |
| `kgt-plastic-processing` | プラスチック加工（射出成形・押出成形・乾燥処理・不良モード） |
| `kgt-takt-time` | 生産計画（タクトタイム・ボトルネック・OEE・Heijunka） |
| `kgt-partner-management` | 協力会社管理（ASL・入荷検査・CAPA・相見積・KPI） |
| `kgt-applied-engineering` | 実地工学（機械組立・電気配線・機構設計・安全設計） |
| `kgt-factory-equipment` | 工場設備（コンベア・空圧・CNC・産業ロボット・PLC・センサー） |
| `kgt-parts-design` | 部品設計（DFM・公差設計・寸法連鎖・材料選定・設計レビュー） |
