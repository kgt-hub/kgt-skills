# 協力会社管理専門エージェント（KGT グループ共通）

製造業・商社業務における協力会社（外注先・仕入先）の管理知識を提供します。

---

## 1. KGT の商社モデル

```
顧客（ANRITSU等）
  ↓ PO（発注）
KGT（商社機能 + 内製工場）
  ├── 内製（In-House）: KGT 工場で切削加工
  └── 外注（Outsource）: 協力会社に加工委託 or 材料調達
        ↓ 協力会社からの納品
KGT → 顧客へ出荷

マージンの発生:
  顧客単価（売値）- 仕入原価（協力会社への支払い）= 粗利
  → 協力会社の品質・納期・価格が直接利益に影響
```

## 2. 協力会社の評価軸

### 3軸評価モデル

```python
SUPPLIER_EVALUATION_AXES = {
    "品質（Quality）": {
        "指標": "入荷不良率（不良品数 / 入荷数）",
        "目標": "< 0.5%",
        "警告": "1〜3%",
        "危険": "> 3%（取引停止を検討）",
        "確認方法": "入荷検査記録・不良通知履歴",
    },
    "納期（Delivery）": {
        "指標": "納期遵守率（期日内納品数 / 全発注数）",
        "目標": "> 95%",
        "警告": "85〜95%",
        "危険": "< 85%（代替先を確保）",
        "確認方法": "発注日vs実際入荷日の記録",
    },
    "価格（Cost）": {
        "指標": "マージン率（売値 - 仕入値）/ 売値",
        "目標": "> 20%（一般部品）",
        "注意": "プレッシャー交渉 vs 長期関係のバランス",
        "確認方法": "相見積もりによる市場価格確認",
    },
}

def calculate_supplier_score(quality: float, delivery: float, cost: float) -> dict:
    """
    総合評価スコア（100点満点）
    """
    score = (quality * 0.4) + (delivery * 0.35) + (cost * 0.25)
    if score >= 80:
        return {"grade": "A（優先使用）", "score": score}
    elif score >= 60:
        return {"grade": "B（条件付き使用）", "score": score}
    return {"grade": "C（要改善 / 取引見直し）", "score": score}
```

## 3. 承認仕入先リスト（ASL: Approved Supplier List）

```
目的:
  ISO 9001 §8.4「外部から提供されるプロセスの管理」に対応
  → 品質・納期・価格を満たした承認済み協力会社のリスト

ASL の内容:
  □ 会社名（タイ語・英語・日本語）
  □ 所在地・連絡先
  □ 取扱品目カテゴリ（旋削・フライス・熱処理・表面処理等）
  □ 認証資格（ISO 9001・IATF 16949等）
  □ 承認日・次回評価予定日
  □ 評価グレード（A/B/C）
  □ 特記事項（独自技術・弱点・交渉経緯）

運用ルール:
  □ 新規仕入先は評価・試作を経てから ASL に追加
  □ 年1回以上の定期評価・更新
  □ C グレード仕入先は改善計画の提出を求める
  □ 重要品目は複数の ASL 仕入先を確保（BCP）
```

## 5. 不良通知と CAPA 要求

```
不良通知のフロー:
  入荷検査で不合格
    ↓
  1. 不良品を隔離（赤タグ）
  2. 仕入先に不良通知を発行
  3. 暫定処置（代替品手配・手直し）
  4. 仕入先からの根本原因分析・是正処置報告を受領
  5. 是正処置の有効性を確認
  6. 再発なければクローズ

仕入先への CAPA 要求書記載事項:
  □ 不良品の写真・サンプル（返却または保管）
  □ 不良内容（寸法・外観・機能）
  □ 納期遅延の影響（生産停止・顧客への影響）
  □ 期限（通常: 受領後5〜10営業日以内）

評価への反映:
  □ 不良発生 → 評価スコアを下げる
  □ CAPA 期限内完了 → 減点を軽減
  □ 再発 → 取引停止または ASL からの降格
```

## 6. 相見積（Competitive Bidding）

```
目的: 市場価格の把握・適正価格での調達・仕入先への競争意識

実施タイミング:
  □ 新規品目の初回発注
  □ 単価が高いと感じる場合
  □ 定期的な価格見直し（年1回推奨）
  □ 仕入先との関係が固定化している場合

見積依頼時の注意:
  □ 同一図面・仕様で複数社に依頼（条件を統一）
  □ 数量・納期条件を明示
  □ 見積有効期限を確認（通常 30〜60 日）

分析の観点:
  単純最安値選択の罠:
    → 実績・品質・納期対応力が不明な業者のリスク
    → 安値の理由（材料品質・手抜き加工）を確認

  総コスト（TCO）での判断:
    調達価格 + 品質不良コスト + 納期遅延コスト + 管理コスト
    → 安くても不良が多ければ TCO では高くなる

価格交渉のアプローチ:
  1. 相見積結果を事実として伝える（プレッシャーをかけすぎない）
  2. 数量増・継続取引を条件に値引き交渉
  3. 協力会社の原価構造を理解した上で着地点を探る
  4. 長期的なパートナーシップを壊さない範囲で交渉
```

## 7. 仕入先との関係管理

```
ランク別対応:

A ランク（優先使用）:
  → 取引量を増やす・新規品目を優先的に依頼
  → 定期訪問・情報共有
  → 将来の増産計画を共有してキャパ確保

B ランク（条件付き使用）:
  → 改善要求を明確に伝える
  → 改善計画の提出を求める
  → 定期レビューで改善度を確認

C ランク（要改善）:
  → 発注量を絞る・代替先を開拓
  → 改善計画の合意・モニタリング
  → 改善なければ ASL からの除外

新規仕入先の開拓:
  □ 定期的な市場調査（工業展示会・商工会議所等）
  □ 既存仕入先からの紹介
  □ 試作品発注 → 品質・納期の確認 → 本採用審査
  □ 工場監査（重要品目は現地視察）

BCP（事業継続計画）の観点:
  □ 単一仕入先への依存を避ける（特に重要品目）
  □ 主仕入先 + 代替仕入先を確保
  □ 仕入先の経営状況・自然災害リスクを定期確認
```

## 8. 協力会社 KPI

```
月次でモニタリングすべき指標:

品質:
  □ 入荷不良率（仕入先別）
  □ 不良通知発行件数（仕入先別）
  □ CAPA 期限内完了率

納期:
  □ 納期遵守率（仕入先別）
  □ 平均遅延日数
  □ 緊急対応件数（至急依頼が多い仕入先は根本問題あり）

価格:
  □ 平均仕入単価の推移（月次）
  □ 値上げ要求件数・承認率

総合:
  □ 仕入先評価スコアの推移
  □ ASL の A/B/C 比率
  □ 集中度（特定仕入先への依存度）
```
