### コンベア設計パラメータ

```
搬送速度の計算:
  必要速度 = 搬送物の間隔 × 1時間の処理数 / 3600秒

モーター出力の選定:
  P = (搬送物重量 + ベルト重量) × 速度 × 摩擦係数 / 機械効率
  安全率: 1.5〜2.0 を掛ける

ベルト張力管理:
  緩すぎ → スリップ・蛇行
  張りすぎ → 軸受過負荷・ベルト寿命短縮
  → テークアップ（張力調整機構）で適切に管理
```

### エアシリンダ

```python
CYLINDER_TYPES = {

        "用途": "ストッパー・クランプ（電源OFF時のバネ復帰が安全）",
    },
    "ガイド付きシリンダ": {
        "動作": "複動 + ガイドロッド",
        "用途": "横荷重がある押出し・精密位置決め",
    },
    "ロータリーアクチュエーター": {
        "動作": "直動→回転変換",
        "用途": "ワーク反転・ターンテーブル",
    },
    "エアグリッパー": {
        "動作": "把持・開放",
        "用途": "ロボットのエンドエフェクター・部品チャック",
    },
}

def calculate_cylinder_force(bore_mm: float, pressure_MPa: float) -> float:
    """シリンダの理論推力（N）"""
    import math
    area = math.pi * (bore_mm / 2) ** 2  # mm²
    force = area * pressure_MPa           # N
    return round(force, 1)

# 例: φ40mm, 0.5MPa → 628N

# シリンダの速度調整: スピードコントローラ（絞り弁）で排気側を絞る
```

### 空圧回路の構成

```
コンプレッサー
  ↓ 除湿（エアドライヤー）
  ↓ 清浄（フィルター）
  ↓ 減圧（レギュレーター 0.3〜0.6 MPa に調整）
  ↓ 給油（ルブリケーター / オイルフリーシリンダには不要）
  ↓ 電磁弁（ON/OFF or 比例制御）
  ↓ スピードコントローラ（速度調整）
  ↓ シリンダ

FRL ユニット: フィルター（F）+ レギュレーター（R）+ ルブリケーター（L）をセット化
```

### CNC プログラムの基礎（G コード）

```
主な G コード:
  G00: 早送り位置決め（非切削移動）
  G01: 直線補間切削
  G02: 円弧補間（時計回り）
  G03: 円弧補間（反時計回り）
  G04: ドウェル（停止時間）
  G41/G42: 工具径補正（左/右）
  G90: 絶対座標
  G91: インクリメンタル（相対座標）

主な M コード:
  M03: 主軸正転（切削開始）
  M05: 主軸停止
  M06: 工具交換（ATC）
  M08: 切削液 ON
  M30: プログラム終了
```

### エンドエフェクター（Hand）

```

  → 平面・滑らか面に有効

  → 吸盤の選定: 搬送物の形状・重量・表面粗さ

機械式グリッパー（Mechanical Gripper）:
  → 不規則形状・重量物
  → エア駆動 or 電動
  → 把持力の計算: 搬送重量 × 安全率 / 摩擦係数

  → 鉄系金属の高速・確実把持

軟質グリッパー（Soft Gripper）:
  → 食品・柔軟物・不規則形状
  → エアで膨らむゴム材料
```

## 6. PLC 制御（シーケンス制御）

```
PLC（Programmable Logic Controller）:
  ラダー図（Ladder Diagram）で順序制御を記述
  → デジタルI/O（センサー入力・アクチュエーター出力）
  → アナログI/O（温度・圧力・速度）

主要メーカー:
  MITSUBISHI（三菱）MELSEC: タイの製造業で最もよく使われる
  OMRON（オムロン）SYSMAC: 日系工場で多い
  SIEMENS SIMATIC: 欧米系工場
  Allen-Bradley（Rockwell）: 自動車系

安全PLC:
  ISO 13849 / IEC 62061 に対応
  安全回路（非常停止・ライトカーテン）の処理に使用
  → 通常の PLC とは別に安全 PLC を使用（または Safety CPU）

デバッグ・保守:
  □ ラダー図のオンライン監視
  □ エラーコードの記録・確認
  □ PLC プログラムのバックアップ（定期実施必須）
  □ バッテリーバックアップの確認（プログラム保持用）
```

## 7. 設備保全（Maintenance）

```
予防保全（PM: Preventive Maintenance）:
  定期的な点検・交換でトラブルを未然に防ぐ
  → 潤滑油交換・ベルト張力確認・フィルター清掃

予知保全（PdM: Predictive Maintenance）:
  振動・温度・電流値を監視して異常を予知
  → IoT センサーで連続監視・AI 異常検知

設備台帳の管理:
  □ 設備名・型式・製造番号・設置日
  □ 主要消耗品と交換周期
  □ 保全履歴（修理内容・費用・担当者）
  □ メーカー連絡先・取扱説明書の保管場所

MTBF と MTTR:
  MTBF（平均故障間隔）= 総稼働時間 / 故障回数 → 高いほど信頼性高
  MTTR（平均修復時間）= 総修復時間 / 故障回数 → 低いほど保全性高
  稼働率 = MTBF / (MTBF + MTTR)
```
