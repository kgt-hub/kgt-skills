## 6. 材料選定フローチャート

```
精密部品用材料選定:

使用温度 > 150°C?
  Yes → PEEK / PPS / PTFE / FKM を検討
  No  ↓

耐薬品性が重要?
  Yes → PTFE / PEEK / PPS / FKM を検討
  No  ↓

寸法精度 < ±0.05mm?
  Yes → POM / PEEK（吸湿考慮）
  No  ↓

耐摩耗・自己潤滑が必要?
  Yes → POM / PA / PTFE を検討
  No  ↓

コスト優先?
  Yes → POM / PA6 / ABS
  No  → 機能優先で選定

シール・ガスケット用:
  耐油 → NBR / FKM
  耐熱 → シリコーン / FKM
  耐候 → EPDM / CR
  超耐熱 → FKM / シリコーン
  コスト優先 → NBR / EPDM
```

## 7. 材料在庫管理の原則

```
プラスチック:
  □ FIFO（先入先出）を徹底
  □ PA/PC は密封保管（防湿袋 + シリカゲル）
  □ ロット番号管理（品質問題発生時のトレーサビリティ）
  □ 保管温度: 室温（直射日光・高温・湿気を避ける）
  □ 有効期限の確認（多くの樹脂は2〜5年）

ゴム・スポンジ:
  □ 直射日光・オゾン（モーター付近）を避ける
  □ 変形しない状態で保管（圧縮・折り曲げ禁止）
  □ NBR: 油から離して保管
  □ 有効期限（ゴムは通常2〜5年）

コンベアベルト:
  □ 芯にかけて丸めて保管（折り目防止）
  □ 直射日光・熱源から遠ざける
  □ 在庫数量は納期を考慮して安全在庫を設定
```
