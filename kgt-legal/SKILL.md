# 法務専門エージェント（KGT グループ共通）

タイと日本の両法域にまたがる製造業・商社業務に関する法務知識を提供します。
具体的な法的判断は弁護士に委ねつつ、実務判断を支援します。

## 適用法域

| 法域 | 主な関連法 | 適用場面 |
|---|---|---|
| タイ | 民商法典・外国事業法・労働保護法・BOI 関連法 | 現地法人の運営・顧客/仕入先との契約 |
| 日本 | 民法・会社法・下請法・製造物責任法（PL法）・不正競争防止法 | 日本本社・日本顧客との取引 |
| 国際 | CISG（国際物品売買条約）・インコタームズ | 国境をまたぐ売買契約 |

## 1. 売買契約・取引基本契約

### タイでの取引契約

```
タイ民商法典（Civil and Commercial Code）の原則:
- 口頭契約も有効（ただし書面化を強く推奨）
- タイ語版が正文となることが多い（争訟時）
- 準拠法: 外国法も選択可能（CISG 適用に注意）

実務チェックリスト:
□ 取引基本契約（MA: Master Agreement）の締結
□ 品質基準・検査方法の明記
□ 支払条件（TT30日・LC等）の明示
□ 不良品の処理手順・責任範囲
□ 秘密保持条項（NDA）
□ 準拠法・管轄裁判所の指定（タイ裁判所 or 仲裁）
```

### 下請法（日本）

```
資本金 3億円超の親事業者が 3億円以下の下請事業者に
製造・修理・役務・情報成果物の委託をする場合に適用。

禁止行為（特に注意）:
- 発注後の単価の一方的引き下げ
- 受領拒否・返品
- 支払遅延（60日以内に支払義務）
- 不当なやり直し要求

対策:
□ 3条書面（発注書）の交付と保存
□ 支払期日の管理（受領日から60日以内）
□ 単価変更は合意書を取得
```

## 2. 製造物責任（PL法 / 製品安全）

```
日本 PL 法（製造物責任法）:
- 製造物の「欠陥」により他者に損害を与えた場合の無過失責任
- タイ製造業者が日本向けに製品を供給する場合も適用対象

製造物責任への対策:
□ 製品仕様・品質基準の明文化
□ 製造記録・検査記録の保存（最低5年推奨）
□ 材料・外注先のトレーサビリティ確保
□ 製品安全に関する顧客への注意書き
□ PL 保険の加入検討

タイ製品責任法（2009年施行）:
- 日本 PL 法に近い構造
- 消費者向け製品が主な対象
```

## 3. 労働法（タイ）

```
労働保護法（Labour Protection Act）の主要事項:

勤務時間:
- 法定労働時間: 1日8時間・週48時間（危険業務は週42時間）
- 時間外手当: 通常賃金の1.5倍（休日出勤は3倍）

有給休取得:
- 勤続1年以上: 年6日以上（会社規則でさらに付与可）

解雇:
- 解雇予告: 1賃金期間以上前（または予告手当）
- 退職金（Severance Pay）:
  120日〜1年未満: 30日分
  1年〜3年未満:   90日分
  3年〜6年未満:  180日分
  6年〜10年未満: 240日分
  10年〜20年未満: 300日分
  20年以上:      400日分

外国人労働者:
- ワークパーミット（Work Permit）必須
- 就労ビザ（Non-B）と連携
- 外国人1人に対してタイ人4人の雇用義務
```

## 4. 会社法・外資規制（タイ）

```
外国事業法（Foreign Business Act: FBA）:
- リスト2（許可が必要な業種）/ リスト3（厳しく規制）
- 製造業は原則 FBA の対象外（外資100%可）
- ただし土地所有は外国人土地法で制限

BOI（投資奨励庁）の優遇措置:
- 法人税免除（最大8年）
- 輸入関税免除（機械・原材料）
- 外国人就労者のビザ・ワークパーミット取得が容易に
- 土地所有権（BOI 企業に限り）

タイ有限会社（บริษัทจำกัด）設立:
- 最低3名の発起人
- 最低資本金（事業許可により異なる）
- タイ人株主51%以上（FBA 適用業種）or 外資100%（製造業）
```

## 5. 知的財産

```
特許（タイ / 日本）:
- 製品・製造方法の保護
- タイ特許庁への出願（thai.ipthailand.go.th）
- 優先権主張: パリ条約（出願後12ヶ月以内）

営業秘密・不正競争防止:
- タイ: 営業秘密法（Trade Secrets Act）
- 日本: 不正競争防止法（刑事罰あり）
- 対策: NDA の締結・アクセス制限・退職時誓約書

商標:
- タイと日本で別途登録が必要
- タイ商標出願: IPTHAILAND への出願
- 先願主義（先に出願した者が権利を持つ）

顧客図面・仕様書の扱い:
- 顧客から受領した図面は顧客の知的財産
- 第三者提供・複製は原則禁止
- 工場内での目的外使用も契約違反リスク
```

## 6. 輸出規制・経済制裁

```
日本の輸出貿易管理令:
- 安全保障上の懸念品目（デュアルユース品）は経済産業省の許可が必要
- 精密部品・工作機械等が対象になる場合がある

対象国・団体リスト:
- 外国為替法に基づく経済制裁（イラン・北朝鮮等）
- 米国 OFAC リスト（間接的に関係する取引にも影響）

確認フロー:
□ 顧客・仕入先の名称を制裁リストと照合
□ 該非判定（Commodity Classification）の実施
□ 該当する場合は経済産業省へ許可申請
□ EAR（米国輸出管理規則）の再輸出規制も確認
```

## 7. 契約書レビューの観点

```
売買・製造委託契約のチェックポイント:

1. 当事者の特定
   - 正式法人名（英語・タイ語・日本語）と所在地
   - 代表者の権限確認

2. 対象物・仕様の明確化
   - 品番・図面番号・改訂版番号
   - 合否判定基準

3. 価格・支払条件
   - 通貨（THB/JPY/USD）
   - 支払方法（TT/LC）・期日
   - 為替リスクの分担

4. 納期・リスク移転
   - インコタームズの明示（EXW/FOB/DAP等）
   - 遅延ペナルティ

5. 品質保証・クレーム
   - 保証期間・保証範囲
   - クレーム通知期限

6. 秘密保持
   - 図面・仕様書・価格情報

7. 契約終了・解除
   - 解除事由・手続き
   - 在庫・仕掛品の処理

8. 準拠法・紛争解決
   - タイ法 or 日本法
   - タイ仲裁院（THAC）/日本商事仲裁協会（JCAA）
```

## 8. よくある法務リスクと対策

| リスク | 原因 | 対策 |
|---|---|---|
| 口頭合意のみで取引開始 | 急ぎの受注・関係への過信 | 受注確認書・注文請書を必ず書面化 |
| 顧客クレームで全損害賠償 | PL リスク・契約上の責任無制限 | 損害賠償上限条項（cap）を設定 |
| 図面流出 | 退職した外注先従業員 | NDA + アクセス制限 + 電子透かし |
| 外国人スタッフの違法就労 | ワークパーミット更新忘れ | 有効期限のカレンダー管理 |
| 下請法違反 | 発注書なしの口頭発注 | 全発注を書面（3条書面）で管理 |
| 退職金の未積立 | タイ労働法の見落とし | 長期在籍者の退職金試算を定期実施 |
