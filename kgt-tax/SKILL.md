# 税務・会計専門エージェント（KGT グループ共通）

タイと日本の両国にわたる製造業・商社の税務・会計知識を提供します。
具体的な申告・納税は税理士・会計士に委ねつつ、実務判断を支援します。

## 適用税制の概要

| 税目 | タイ | 日本 |
|---|---|---|
| 法人税 | 20%（中小企業優遇あり）| 実効税率 約30%（法人税+地方税）|
| 付加価値税 | VAT 7% | 消費税 10% |
| 源泉徴収税 | 配当3%・利子・サービス等 | 支払調書制度 |
| 移転価格 | 関連者間取引の独立企業間価格 | 同左 |

## 1. タイの税務

### 法人税（Corporate Income Tax: CIT）

```
標準税率: 20%

中小企業優遇（以下をすべて満たす場合）:
  - 払込資本金 ≤ 500万バーツ かつ
  - 年間売上 ≤ 3,000万バーツ
  税率:
    〜30万バーツ:    0%
    30万〜300万バーツ: 15%
    300万バーツ超:   20%

BOI 優遇:
  法人税免除（最大8年）+ 輸入関税免除
  → BOI 企業は対象期間中の CIT が 0%

申告・納付:
  中間申告: 事業年度終了後6ヶ月以内（推定税額の50%）
  確定申告: 事業年度終了後150日以内
```

### VAT（付加価値税）

```
税率: 7%（2026年3月時点の暫定適用税率）
標準税率は 10% だが長年7%に据え置き

登録義務: 年間売上 180万バーツ以上で VAT 登録必須

インボイス制度:
  Tax Invoice（税額票）を発行・保存
  仕入 VAT（Input Tax）と 売上 VAT（Output Tax）の差引納付

輸出:
  ゼロ税率（0%）が適用される
  → 製品をタイから輸出する場合、VAT 還付申請が可能

源泉税（Withholding Tax: WHT）:
  サービス料・賃借料・配当・利子等の支払時に源泉徴収
  主な税率:
    サービス（法人間）: 3%
    賃借料:           5%
    配当（法人→法人）: 10%
    配当（法人→個人）: 10%
  → 支払翌月7日（電子:15日）までに納付
```

### 会計基準（タイ）

```
Thai Financial Reporting Standards（TFRS）:
  IFRS をベースとして制定
  中小企業向け: Thai GAAP（Simple accounting）

決算期:
  タイ法人の事業年度は自由に設定可能
  一般的に1月〜12月 or 4月〜3月

会計記録の保存:
  会計帳簿・証票: 5年間保存義務
  税務上の書類: 5年間（延長される場合あり）
```

## 2. 日本の税務

### 法人税・地方税

```
実効税率: 約30〜34%（法人税率23.2% + 地方法人税 + 住民税・事業税）

中小法人優遇（資本金1億円以下）:
  課税所得 800万円以下: 法人税率 15%
  課税所得 800万円超:   法人税率 23.2%

申告・納付:
  中間申告: 事業年度開始から6ヶ月以内
  確定申告: 事業年度終了後2ヶ月以内
  （延長申請で最長3ヶ月）
```

### 消費税

```
税率:
  標準税率: 10%
  軽減税率: 8%（飲食料品・新聞）

インボイス制度（2023年10月〜）:
  適格請求書（インボイス）発行登録が必要
  登録番号を請求書に記載
  未登録業者からの仕入は仕入税額控除が段階的に不可に

輸出:
  ゼロ税率（免税）
  輸出証明書類の保存が必要
```

## 3. 移転価格税制（Transfer Pricing）

```
対象: 国外関連者（グループ内の海外法人）との取引

タイ↔日本間の内部取引例:
  - タイ製造子会社 → 日本親会社への製品販売
  - 日本親会社 → タイ子会社への管理費配賦
  - ロイヤルティ・技術指導料

独立企業間価格（Arm's Length Price）の検証:
  比較可能非関連者価格法（CUP）
  再販売価格基準法（RPM）
  原価基準法（CPM）
  取引単位営業利益法（TNMM）

文書化義務:
  タイ: 年間関連者間取引額が閾値超で Local File 提出
  日本: 一定規模以上のグループで国別報告書（CbCR）
```

## 4. 製造業に特有の税務処理

### 材料・在庫の会計処理

```
在庫評価方法（タイ・日本共通で選択）:
  先入先出法（FIFO）: 価格変動時に利益が変動
  移動平均法: 価格の平滑化効果
  総平均法: 期末一括計算

製造原価の構成:
  直接材料費 + 直接労務費 + 製造間接費 = 製造原価
  → 在庫として B/S に計上し、出荷時に売上原価に振替

不良品・廃棄の処理:
  廃棄損: 損益計算書に費用計上
  保険求償が可能な場合は保険収入と相殺
```

### 固定資産・減価償却（タイ）

```
法定耐用年数（タイ歳入局の告示に基づく）:
  建物:         20年（5%/年）
  機械設備:      5年（20%/年）
  コンピュータ:  3年（33.3%/年）
  工具・金型:   3〜5年

定率法・定額法のいずれも選択可能
BOI 設備は加速償却の優遇あり（別途確認）
```

### 外注費・加工費の源泉徴収

```
タイ国内の法人へのサービス料支払時:
  → 3% 源泉徴収して翌月7日までに納付
  → 支払相手（協力会社）に源泉徴収証明書（PND53）を交付

個人への支払（フリーランス等）:
  → 3% 源泉徴収（PND3 で申告）

外国法人へのサービス料（国境を越える場合）:
  → 15% 源泉徴収（租税条約で軽減される場合あり）
  → 日本法人への支払は日タイ租税条約（5%/10%等）を確認
```

## 5. 日タイ租税条約

```
正式名称: 「所得に対する租税に関する日本国と
           タイ王国との間の条約」（1990年発効）

主な内容:
  配当:     源泉地国 20% → 条約適用で 10%（親子会社: 10%/15%）
  利子:     源泉地国 25% → 条約適用で 10%/15%
  使用料:   源泉地国 15% → 条約適用で 15%
  役員報酬: 居住地国での課税
  PE（恒久的施設）認定要件

手続き:
  タイへの支払時に条約適用を受けるには
  タイ歳入局への届出（Form RD 0702）が必要
```

## 6. 税務リスク管理

```
よくある指摘事項（税務調査）:

タイ:
  □ VAT の過少申告（売上の計上漏れ）
  □ 源泉徴収の計上もれ・申告漏れ
  □ 関連者間取引の価格の合理性
  □ 経費の業務関連性の証明（领収书保存）

日本:
  □ 海外子会社への過大な費用配賦
  □ タコス費用・交際費の損金不算入
  □ 移転価格文書化の不備

対策:
  □ 契約書・請求書・証票類の完全保存（5年）
  □ 源泉徴収の翌月申告を自動リマインド
  □ 期中での VAT 申告書と帳簿の突合
  □ 関連者間取引の価格根拠を文書化
```

## 7. 財務 KPI と税務の関係

```
EBITDA（税引前・利息・償却前利益）:
  = 売上高 - 売上原価 - 販管費 + 減価償却費
  → 事業の稼ぎを評価する基本指標

実効税負担率:
  = 法人税等 / 税引前当期純利益
  → タイ: 目標 20%未満（BOI 優遇活用）
  → 日本: 実効税率 約30%

キャッシュフロー注意:
  利益が出ていても VAT・源泉税の納付で CF が圧迫される
  → 毎月の税金キャッシュアウトを資金繰り計画に含める
```
