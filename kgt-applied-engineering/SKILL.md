## 1. 機械組立（Mechanical Assembly）

### 組立精度管理

```
基本的な組立誤差の種類:
  寸法誤差の累積（寸法連鎖）:
    → 個々の部品公差が積み重なって組立体の誤差が大きくなる
    → 最悪ケース解析（Worst Case Analysis）で設計段階から管理

嵌め合いの種類と用途:
  すき間嵌め（Clearance Fit）: 回転・摺動部（軸受・スライド）
    → 常に隙間がある → 回転・摺動がスムーズ
  中間嵌め（Transition Fit）: 位置決め（ノックピン・ダウェルピン）
    → 圧入または軽い打ち込みで固定
  しまり嵌め（Interference Fit）: 圧力固定（ブッシュ・プーリー圧入）
    → 解体を想定しない場合に使用
```

### 接着・溶接による結合

```
接着剤の選定:
  エポキシ系:    金属・プラスチック・セラミックの接着。高強度。加熱で硬化促進
  シリコーン系:  耐熱・耐振動。防水シールにも使用。
  アクリル系:    プラスチック同士の透明接着（PC・アクリル）
  嫌気性接着剤:  ネジのゆるみ止め（Loctite 等）・圧入部の固定

プラスチックの溶着:
  超音波溶着: ABS・PC・POMなどの同種材料接合。高速・量産向け
  熱板溶着: ポリエチレン・ポリプロピレン
  溶剤溶着: アクリル・PC・ABS（溶剤塗布で接合面を溶かして圧着）
  注意: 異種材料の溶着は原則不可
```

## 2. 電気・電気配線（Electrical）

### 基本安全事項

```
感電防止:
  作業前に必ずブレーカーをOFF（ロックアウト・タグアウト）
  検電器で無電圧を確認してから作業
  200V以上: 複数人での作業を推奨

アース（接地）:
  設備フレームのアース接続は法的義務（電気設備技術基準）
  → 設備の絶縁不良時に人体への電流を迂回
  → 接地抵抗: D種接地（100Ω以下）

絶縁:
  電線の絶縁被覆の状態を定期点検
  ケーブルトレイの曲げR制限を守る
  → 過度の曲げで芯線が折れる・被覆が割れる
```

### 制御盤（Control Panel）

```
配電系統:
  電源入力 → メインブレーカー → 各機器用ブレーカー → 機器

主な電気機器:
  PLC（シーケンサー）: シーケンス制御の中枢
  サーボアンプ: サーボモーターの速度・位置制御
  インバーター: AC モーターの回転数制御
  リレー: 小電流で大電流をON/OFF
  タイマー: 時間制御

安全回路:
  非常停止（EMO）ボタン: 全電源（またはサーボ）を遮断
  インターロック: ドア開でサーボ停止等
  安全リレー（Safety Relay）: 安全規格（ISO 13849）に対応
```

### 電気安全規格

```
国際規格:
  IEC 60204-1: 機械の電気安全（配線・制御盤設計の基本規格）
  ISO 13849: 機械安全（制御システムの安全性能評価）
  IEC 61508: 機能安全（SIL レベル）

日本の規格:
  電気設備技術基準（電技）
  内線規程（JEAC 8001）

タイの規格:
  TISI（タイ工業標準）/ IEC 準拠

制御盤の IP 等級（保護等級）:
  IP54: 防塵・飛水（工場標準）
  IP65: 防塵・防噴流（洗浄水がかかる環境）
  IP67: 防塵・水没（屋外・湿潤環境）
```

### 軸受（Bearing）の選定

```
転がり軸受（Rolling Bearing）:
  深溝玉軸受（6000系）: 一般的な高速回転
  アンギュラ玉軸受（7000系）: ラジアル + アキシャル荷重。スピンドル
  円すいころ軸受:       大きなアキシャル荷重。ホイールハブ
  自動調心球軸受:       軸のたわみや傾きを吸収

滑り軸受（Plain Bearing / Bushing）:
  材質: 銅合金・PTFE・プラスチック
  特徴: 構造簡単・衝撃に強い・無給油も可（PTFE）
  用途: 低速・往復運動・小型機構

選定基準:
  高速・低荷重 → 深溝玉軸受
  高荷重・低速 → 円すいころ軸受または滑り軸受
  往復運動 → 直動ガイド（LM ガイド）
  無給油・小型 → プラスチックブッシュ（POM・PTFE）
```

### 防振・緩衝

```
防振ゴムの設計:
  固有振動数を設備の励起振動数の 1/√2 以下に設定
  → 一般的に防振ゴムは 5〜20Hz に固有振動数を設定

緩衝材（ショックアブソーバー）:
  油圧ダンパー: 高精度な衝撃吸収・速度比例型
  空圧ダンパー: 軽量・調整可能
  ウレタン / ゴムバンパー: 安価・短ストローク用

用途別選定:
  搬送装置の位置決め → 調整式ショックアブソーバー
  設備の脚部防振 → 防振ゴムパッド
  ストッパー衝突緩衝 → ゴムバンパー・ウレタン
```

## 4. 安全設計（Safety Engineering）

```
機械安全の基本原則（ISO 12100）:

1. 本質的安全設計（危険を取り除く）
   → 鋭利なエッジを除去・危険エリアに近づけない設計

2. 安全防護（残留リスクを遮断）
   → ガード・インターロック・光カーテン

3. 使用上の情報（残留リスクを伝える）
   → 警告ラベル・操作マニュアル

リスクアセスメント（RA）:
  1. 機械の使用制限を定める
  2. 危険源を特定する
  3. リスクの見積もり（重篤度 × 発生確率 × 回避可能性）
  4. リスク評価（許容可能か）
  5. リスク低減の実施

安全距離（ISO 13857）:
  光カーテン設置距離 = 手の到達速度（1600mm/s）× 停止時間(s) + 検出距離
```
